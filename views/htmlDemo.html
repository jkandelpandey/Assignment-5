New Semantic Elements

HTML5 introduced several new semantic elements that provide better structure and meaning to web content. These include:

  • <header>: Represents introductory content or a set of navigational links.
  • <nav>: Defines a set of navigation links.
  • <article>: Represents a self-contained piece of content that could be distributed independently.
  • <section>: Defines a section in a document.
  • <footer>: Represents footer content, typically containing information about the author or copyright information.

Multimedia Elements

HTML5 makes it easy to add multimedia content to web pages. Some of the new multimedia elements include:

  • <audio>: Used to embed sound content.
  • <video>: Used to embed video content.
  • <canvas>: A container for graphics, which you can draw with JavaScript.

Enhanced Forms

HTML5 has brought new attributes and input types that make form validation and user input easier and more reliable:

  • placeholder: Specifies a short hint that describes the expected value of an input field.
  • required: Specifies that an input field must be filled out before submitting the form.
  • email and url: New input types that automatically validate the entered data.
